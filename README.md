Java Factorial and Stack Operations Homework

This repository contains Java programs that demonstrate:

Factorial calculation using different looping structures (for, while, and do...while).

Recursive factorial calculation and performance comparison with iterative methods.

Stack operations using various data types and custom objects.
| File                         | Description                                                                        |
| ---------------------------- | ---------------------------------------------------------------------------------- |
| `FactorialUsingFor.java`     | Calculates factorial using a **for loop**.                                         |
| `FactorialUsingWhile.java`   | Calculates factorial using a **while loop**.                                       |
| `FactorialUsingDoWhile.java` | Calculates factorial using a **do...while loop**.                                  |
| `FactorialComparison.java`   | Compares factorial calculation performance for **iterative vs recursive** methods. |
| File                                  | Description                                                                                                                                                 |
| ------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `Item.java`                           | Defines an `Item` class with `name` and `value` fields.                                                                                                     |
| `DiverseStackOperationsWithItem.java` | Performs **push**, **peek**, **pop**, and **empty checks** on stacks containing different data types: `String`, `Integer`, `Boolean`, `Double`, and `Item`. |
